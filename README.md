# README #

**Rails version : 4.2.1**

### How to start the application ###

`$ bundle exec rake db:migrate RAILS_ENV=development`

`$ bundle exec rake db:seed`

`$ Rails server`

### How to run the tests ###

`$ bundle exec rake db:migrate RAILS_ENV=test`

`$ bunlde exec rspec`
